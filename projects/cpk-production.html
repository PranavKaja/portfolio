CPK PRODUCTION PLANNER

Paste a commissary kitchen's daily order report and get back the make list and per-station pick sheets across 36 campus locations — what used to be manual counting across 100+ items. Runs entirely in your browser; nothing is sent anywhere.

HOW IT WORKS

  • Paste the day's raw order report — each line is one location ordering N of an item.
  • Make list aggregates every order into the total to produce per item.
  • Station grid pivots the orders into an item × location matrix with row/column totals, so each prep station knows exactly how many of each item goes to each café.
  • Routing, bread & ingredient prep (next) explode the counts into label/destination routing, bread to defrost, and raw ingredient quantities via a recipe BOM.

Replaces a manual spreadsheet workflow for the Worcester central production kitchen (UMass Dining). Sample data shown is anonymized.
